User login and registration app using Nodejs | Express | Passport | Mongoose | EJS and some other packages.

Run commands:

Usage                                                                                                                                                            
$ npm install                                                                                                                            
$ npm start
//Run with Nodemon
$ nodemon run

//Visit http://localhost:8000                                                                              
MongoDB
Open "config/keys.js" and add your MongoDB URI, local or Atlas
